# Detecção Facial
Script em python que ultiliza o OpenCV para fazer o a detecção facial
## Pre-requisitos
```
python version 2.7
```
Ira precisar do OpenCV
```
pip install opencv-python
```
### Como usar
```
cd deteccaoDeRosto
cd Face Detection
python Detect_Face.py
```
